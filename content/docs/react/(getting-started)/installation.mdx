Use one of the following methods to install **MijnUI Components**:

- [**NPM Installation**](/docs/react/installation/npm)  
   Fast setup. Suitable when you want updates managed through the package manager.

- [**Manual Copy & Paste**](/docs/react/installation/manual)  
   **Recommended** for full control and long-term stability. This method avoids unexpected changes from future package updates.

Choose the method that matches your workflow. Each option has a dedicated page with details.
